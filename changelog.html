Version 0.1.4

July 18, 2025

Major Agent Capabilities Enhancement

Improved agentic capabilities to be more efficient and powerful. The agent can now perform a wide range of file system operations, execute commands, and leverage tools more effectively.

  • Create files with content
  • Create folders
  • Edit files (with multiple methods)
  • Execute terminal commands
  • Grep codebase for patterns
  • Search codebase semantically
  • Utilize custom MCP tools
  • List directory contents
  • Search the web for information

Version 0.1.3

Intelligent Task Planning

The agent now plans app architecture and generates a to-do list to break down complex tasks and workflows, providing a clear path to completion.
